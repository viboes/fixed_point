Function template to_ureal_t

boost::fixed_point::to_ureal_t

Synopsis

// In header: <boost/fixed_point/number.hpp>


template<int Times, int Resolution> 
  constexpr ureal_t< LOG2(ABS(Times+1)), Resolution > to_ureal_t();

Description

ureal_t compile time factory.

Template Parameters:

- Times: an int representing the number of times.

- Resolution: the binary resolution.

Returns: an ureal_t enough large to represent Times*(2^Resolution).
