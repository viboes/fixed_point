Struct template add_result

boost::fixed_point::add_result

Synopsis

// In header: <boost/fixed_point/number.hpp>

template<typename T1, typename T2 = T1, bool B1 = is_open<T1>::value, 
         bool B2 = is_open<T2>::value> 
struct add_result {
};

Description

Add type metafunction.

The result type depends on whether the types are open/closed.

  • Both are closed: The nested typedef type is only defined if is_same<T1,T2> and is T1.

  • One of them is Open:

    • if one of them is signed : real_t<MAX(R1,R2)+1, MIN(P1,P2), DT(RP1,RP2), DT(OP1,OP2), DT(F1,F2)>.

    • if both are unsigned : ureal_t<MAX(R1,R2)+1, MIN(P1,P2), DT(RP1,RP2), DT(OP1,OP2), DT(F1,F2)>.
