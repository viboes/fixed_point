Function template number_cast

boost::fixed_point::number_cast

Synopsis

// In header: <boost/fixed_point/number.hpp>


template<typename From, typename To> To number_cast(From const & from);

Description

Explicit conversion between fixed_point numbers.

Template Parameters:

- From: the source type of the conversion.

- To: the target type of the conversion.

Parameters:

- from: the number from which the conversion is done.

Requires: From is a fixed_point number or a builtin type and To is a fixed_point number.

Returns: the conversion with possible reduced range and loss of resolution.
